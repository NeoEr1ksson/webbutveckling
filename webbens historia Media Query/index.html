W W W

WWW står för World Wide Web. WWW är ett system som visar länkar,document,websidor och allt som har ladats upp nättet som representerad av URL (en address) en sträng av text som berätar för ett system hur man kan hitta dess doc,länker Etc.

U R L

URL är en förkoting av Uniform Resource Locator, URL gör så webläsaren vet vart och hur den ska öpnna en länk.

H T T P

HTTP står för HyperText Trensfer Protocol, Http är en kommunikationsprotpkoll med deras syfte att över föra HTML-sidor från och till webbens servers till webbens klienter

D N S

DNS står för Domian Name System, DNS "översätter domännamn till IP-adresser" när man skriver in en webbadress(URL) kopplar Dns webbadressen (URL) med IP-adressen av serven som webbsidan är på.

Vilken var Svergies första webssida
